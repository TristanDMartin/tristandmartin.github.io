Audio Routing with Wave Link

Master advanced audio routing for professional streaming setups

Wave Link by Elgato is a virtual audio mixer built specifically for streamers, podcasters, and creators using the Elgato Wave series microphones. It lets you control, mix, and route multiple audio sources to different outputs—offering precise control over what your stream hears vs. what you hear.

If you're aiming for a pro-level, dual-mix setup without the complexity of hardware mixers like GoXLR or Voicemeeter, Wave Link is one of the cleanest solutions available.

What You Need

Hardware

  • Elgato Wave:1, Wave:3, or Wave XLR (required for Wave Link to function)
  • Headphones (for zero-latency monitoring)

Software

  • Wave Link Software
  • Streaming Software: OBS Studio / Streamlabs Desktop

Core Features

  • Nine audio input channels, including Game, Music, Browser, Voice Chat, Mic, etc.
  • Two independent outputs: Monitor Mix (your ears) and Stream Mix (your stream)
  • Per-source volume control for each output
  • Drag-and-drop channel arrangement
  • Integrates with Stream Deck for macro-level control

Setup Instructions

Install and Launch Wave Link
  • Download and install from the Elgato website.
  • Connect your Wave microphone or XLR device.
Assign Audio Sources
  • Open Wave Link and assign applications (game, browser, Discord, music, etc.) to the virtual channels:
  • Set System Sounds to Wave Link System
  • Set Spotify to Wave Link Music
  • Set Discord/Chat apps to Wave Link Voice Chat
  • Set Mic input to the hardware-connected Wave device
Customize Your Mixes
  • Each channel has individual volume sliders for:
  • Monitor Mix: What you hear in your headphones
  • Stream Mix: What your audience hears on stream
  • You can mute, solo, or adjust each input without affecting the other
Configure OBS
  • In OBS, set Mic/Aux Input to Wave Link Stream.
  • Mute all other audio devices in OBS to prevent duplicates.
  • Everything you mix in Wave Link's Stream Mix will now be sent to your broadcast.

Advanced Use Cases

Discord: Mute chat for stream, but keep it in your ears

  • Mute Voice Chat input from Stream Mix, keep it in Monitor Mix

Music-only in your headphones

  • Mute Music input from Stream Mix, keep in Monitor Mix

Instant audio toggles with Stream Deck

  • Use the Elgato Stream Deck to mute, solo, or adjust volume of Wave Link inputs

Dual PC Setup

  • Route Wave Link Stream Mix via Line-Out or virtual cable to the streaming PC
