NDI Configuration

Learn how to set up NDI for advanced streaming workflows

NDI (Network Device Interface) is a protocol developed by NewTek that allows you to send high-quality, low-latency video and audio over your local network. It's a powerful alternative to traditional capture cards—perfect for dual-PC streaming or managing multiple video sources wirelessly or over Ethernet.

What You Need

Software

  • OBS Studio on both PCs
  • NDI Plugin for OBS - Download from GitHub – Palakis OBS NDI Plugin
  • NDI Runtime Installer (required for the plugin to work)

Hardware

  • Two network-connected PCs (wired is strongly recommended for best results)
  • Gigabit Ethernet preferred for smooth, high-bitrate video transfer
  • Compatible router/switch that can handle high throughput

Setup Instructions

  • Install NDI Plugin on Both PCs
    • Download the latest OBS-NDI plugin.
    • Install it on both your Gaming PC (source) and Streaming PC (receiver).
    • Also install the NewTek NDI Runtime when prompted.
  • Enable NDI Output on the Gaming PC
    • Launch OBS Studio on the Gaming PC.
    • Go to Tools > NDI Output Settings.
    • Enable Main Output and give it a name like "GamingPC-OBS".
    • Make sure your gameplay is showing in a scene as normal.
  • Receive NDI Source on Streaming PC
    • Open OBS Studio on the Streaming PC.
    • Add a new NDI Source from the source menu.
    • Select the NDI stream named "GamingPC-OBS".
    • The gameplay should now appear live on your Streaming PC with audio and video.
  • Audio Routing Options
    • NDI transmits audio alongside video by default.
    • Make sure Desktop Audio in OBS on the Gaming PC is routed into the active scene.
    • To avoid audio sync issues, use a Network Sync Offset or adjust in OBS's sync settings.

Sync and Performance Tips

  • Use wired Ethernet (1Gbps or more) for both PCs to minimize latency and packet loss.
  • Match FPS/resolution on both OBS instances for a smoother signal.
  • Disable preview on Gaming PC's OBS to save CPU/GPU.
  • Consider lowering NDI bandwidth mode in the source settings (Low/Medium/High).

When to Use NDI Instead of a Capture Card

Use Case Capture Card NDI
High visual fidelity ✅ Yes ✅ Yes
Wireless or networked workflows ❌ No ✅ Yes
Lowest latency (competitive gaming) ✅ Yes ⚠️ Moderate
Multiple PC / multi-cam production ✅ Complex ✅ Simplified
No extra hardware ❌ Needs card ✅ Software-only

Final Tips

  • NDI is CPU and network intensive, so optimize both your OBS scenes and network infrastructure.
  • NDI works across platforms—great for connecting Mac, PC, or even mobile devices.
  • For mobile use, apps like NDI HX Camera can turn your phone into a webcam on your network.
